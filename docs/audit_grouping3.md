## 4. Санити fuzzy

### Статистика
121 канал (из 1626 с auto) — привязка чисто fuzzy (город НЕ найден в name/username даже ILIKE). Это ~7%.

### Порог
`app/userbot/poller.py:1237`: `threshold = 0.95 if len(city_name) < 5 else 0.85`

### Примеры fuzzy-матчей

**Корректные (большинство):**
```
@chat_nizhny_novgorod → Нижний Новгород (fuzzy "nizhny" vs "Nizhny")
@forum_pangan → Пханган (fuzzy "pangan" vs "Phangan")
@rostov_race_chat → Ростов-на-Дону (fuzzy "rostov" vs "Rostov")
@phiphi_chat → Пхи-Пхи (fuzzy "phiphi" vs "Phi Phi")
@chat_vladivostox → Владивосток (fuzzy "vladivostox" vs "Vladivostok")
```

**Сомнительные (false positives):**
```
@ethpad_chats "Nnnjjj" → Нижний Новгород (fuzzy на "Nnnjjj"?)
@NNR_chat "Николаевская Народная Республика" → Нижний Новгород ("ННР" vs "Нижний"?)
@nnw_chat "Neural Chat" → Нижний Новгород ("nnw" fuzzy на что?)
@nnmidletschat "nnproject MIDlets Chat" → Нижний Новгород ("nn" vs "Nizhny"?)
```

**Оценка:** ~5-10 ложных срабатываний из 121 fuzzy-only каналов (≈5-8%). Короткие слова/аббревиатуры с буквами "nn" приклеиваются к Нижнему Новгороду. В масштабах 2522 каналов — погрешность <0.5%.
