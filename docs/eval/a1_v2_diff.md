## Post-review фиксы (12.07, phase-review фазы A)

Перегон diff после фиксов (origin-split, см. ниже): head 1063 → 174 (16%), gained 0,
liked_lost — те же 4 аренды, все по-прежнему матчатся (3 × scooter-rental, 1 × moto-instructor).

Найдено ревью и исправлено:
1. **Происхождение keyword вместо POS-эвристики:** pymorphy тегает транслит-бренды глаголами
   («дукати», «хендай» — подтверждено на живой БД) → synonym-одиночка самоподтверждалась через
   match_near (дистанция 0). Теперь CompiledKeyword.from_synonym: synonym-одиночки никогда не
   триггерят Pass 1 сами (POS не важен); demand-глаголы гейтируются match_near; demand-существительные
   не гейтируются (латентная ловушка «выкуп» закрыта). Без pymorphy demand-глаголы деградируют
   к старому поведению (recall сохраняется), synonym-гейт работает без морфологии.
2. **Карантин (A3) не глушит личные keywords:** полностью карантинный матч теперь проверяется
   _matches_personal_keyword — при хите уходит в _dispatch с пустыми сегментами (keyword-ветка).
3. **Знаменатель fail-open (A2):** счётчики перенесены в validate_batch — только сообщения,
   реально ходившие к LLM (skip_llm/high-conf не разбавляют rate).
4. **Алерт fail-open:** только llm_mode=blocking (в shadow — ложная паника) + фолбэк на предыдущий
   час при <20 валидаций (слепое пятно на границе часа).
